Abstract#==#We present new/s/leak, a novel tool developed for and with the help of
journalists, which enables the automatic analysis and discovery of newsworthy
stories from large textual datasets. We rely on different NLP preprocessing
steps such named entity tagging, extraction of time expressions, entity
networks, relations and metadata. The system features an intuitive web-based
user interface based on network visualization combined with data exploring
methods and various search and faceting mechanisms. We report the current state
of the software and exemplify it with the WikiLeaks PlusD (Cablegate) data.
